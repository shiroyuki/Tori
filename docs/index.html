Tori, Web Framework for People with Deadline

The common problem of web frameworks is that even if the framework is dead simple to use, deploying sometimes won't be as easy as developing.

Introduction

Tori was forked from one of four main modules of Yotsuba Project. Its objectives are:

  • Simplify and minimize the setup and deploying process for tiny-to-large-scale web development.
  • Based on top of Facebook's open-source Tornado Web Framework.
  • Support the deployment as a standalone WSGI server, a WSGI application for Apache 2 (with mod_wsgi) and a Google App Engine application.
